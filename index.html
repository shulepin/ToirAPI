    Events

    OnLoad()

    • Called when a script is loaded.
    function OnLoad()
    __PrintTextGame("OnLoad works")
end

    OnTick()

    • Trigger every tick.
    function OnTick()
    __PrintTextGame("OnTick works")
end

    OnUpdate()

    • Trigger every screen updates.
    function OnUpdate()
    __PrintTextGame("OnUpdate works")
end

    OnDraw()

    • Trigger every screen updates.
    function OnDraw()
    __PrintTextGame("OnDraw works")
end
